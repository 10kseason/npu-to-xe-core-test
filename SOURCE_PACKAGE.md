# Source Package 1.0

This folder is a source-only copy of the current project state for sharing or archival.

Included:

- root Python package sources under `src/`
- tests, tools, docs, and shaderpacks
- Fabric mod sources under `fabric-npu-bridge-mod/`
- Gradle wrapper files needed to build the Fabric mod
- top-level project files such as `README.md`, `pyproject.toml`, and `start-npu-bridge.bat`

Excluded on purpose:

- `build/` outputs
- caches and temporary directories
- local skill folders
- copied/backup directories
- git metadata

English:

- Use `baePo/1.0/` if you want the ready-to-run distribution package.
- Use `baepo-src/1.0/` if you want the editable source tree.

한국어:

- 바로 실행할 배포 패키지가 필요하면 `baePo/1.0/`를 사용하세요.
- 수정 가능한 소스 트리가 필요하면 `baepo-src/1.0/`를 사용하세요.
